_aspectify

  • _aspectify(name, entity, buffer, cleanName?): void

  • Transforms an entity or CDS aspect into a JS aspect (aka mixin). That is, for an element A we get:

    • the function A(B) to mix the aspect into another class B
    • the const AXtended which represents the entity A with all of its aspects mixed in (this const is not exported)
    • the type A to use for external typing and is derived from AXtended.

    Parameters

    • name: string

      the name of the entity

    • entity: any
    • buffer: Buffer

      the buffer to write the resulting definitions into

    • Optional cleanName: null | string

      the clean name to use. If not passed, it is derived from the passed name instead.

    Returns void

isSelfReference

  • isSelfReference(entityName): boolean

  • A self reference is a property that references the class it appears in. They need to be detected on CDS level, as the emitted TS types will try to refer to refer to types via their alias that hides the aspectification. If we attempt to directly refer to this alias while it has not been fully created, that will result in a TS error.

    Parameters

    • entityName: string

    Returns boolean

    true, if entityName refers to the surrounding class

    Example

    class TreeNode {
      value: number
      parent: TreeNode // <- self reference
    }
